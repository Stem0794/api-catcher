How to generate your GitHub PAT

  1. Go to GitHub → Settings → Developer settings → Personal access tokens → Fine-grained tokens (or use Tokens (classic)).
  2. Click "Generate new token".
  3. Required scope (Classic token): select only gist.
    Required permission (Fine-grained token): under Account permissions, set Gists to Read and write.
  4. Set an expiration date that suits your workflow.
  5. Click "Generate token", copy it, and paste it above.
